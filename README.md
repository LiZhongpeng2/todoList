# TodoList
A modern, VS Code-styled todo list application built with vanilla JavaScript.
## Features
- ✨ Add/Delete tasks
 🔄 Toggle task status
 🔍 Filter tasks (All/Active/Completed)
 📊 Sort by date or priority
 📈 Real-time statistics
 💅 VS Code-inspired design
